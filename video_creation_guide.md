# NudgeAI Video Creation Guide

## Overview
This guide helps you create the actual 2-minute video walkthrough for your NudgeAI project using the files generated by the script.

## Files Generated
- `video_output/nudgeai_walkthrough.html` - Interactive video player with captions
- `video_output/nudgeai_captions.srt` - Subtitle file for video editing
- `video_output/walkthrough_script.md` - Complete narration script
- `video_output/README.md` - Documentation and usage instructions

## Step-by-Step Video Creation

### Option 1: Screen Recording (Recommended)
1. **Open the HTML file** in your web browser
2. **Start screen recording** using software like:
   - OBS Studio (free)
   - Camtasia
   - QuickTime (Mac)
   - Windows Game Bar
3. **Navigate through your NudgeAI project** while narrating using the script
4. **Record for approximately 2 minutes**
5. **Stop recording and save as MP4**

### Option 2: Professional Video Editing
1. **Record separate clips** of your project walkthrough
2. **Import the SRT captions** into your video editing software
3. **Add background music** (optional)
4. **Sync narration** with the script provided
5. **Export final video as MP4**

## Recording Script
Use the script in `walkthrough_script.md` for your narration. The script is timed for approximately 2 minutes.

## Technical Specifications
- **Resolution**: 1920x1080 (1080p)
- **Format**: MP4
- **Duration**: ~2 minutes
- **Audio**: Clear narration
- **Captions**: Use provided SRT file

## Tips for Success
1. **Practice the script** before recording
2. **Use a good microphone** for clear audio
3. **Ensure good lighting** if showing yourself
4. **Keep background noise minimal**
5. **Test your recording setup** first
6. **Speak clearly and at a moderate pace**

## Post-Production
1. **Edit the video** to remove mistakes
2. **Add the SRT captions** for accessibility
3. **Include background music** (optional, low volume)
4. **Export in high quality**
5. **Place the final MP4 in the video_output folder**

## Final Output
Your final video should be named `nudgeai_walkthrough.mp4` and placed in the `video_output` folder alongside the other generated files.

## Troubleshooting
- **Audio issues**: Check microphone settings and background noise
- **Video quality**: Ensure proper resolution and frame rate
- **Timing**: Practice to match the 2-minute duration
- **Captions**: Verify SRT file syncs correctly with video
