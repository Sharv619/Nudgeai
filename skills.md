# NudgeAI Skills

NudgeAI is a sophisticated personal productivity assistant powered by the Model Context Protocol (MCP), Retrieval-Augmented Generation (RAG), and Hugging Face models. Below are its primary skills and capabilities.

## 1. Multi-Source Data Integration
NudgeAI can ingest and process data from multiple personal sources:
- **Google Calendar**: Fetching and analyzing meetings, appointments, and availability.
- **Google Drive**: Indexing and searching through documents for contextual information.
- **Location History**: Tracking and analyzing movement patterns to understand routines.
- **Google Fit**: Monitoring fitness activities, steps, and health metrics.
- **Local Codebase**: Analyzing and optimizing its own source code or other local projects.

## 2. Advanced Retrieval (RAG)
- **Universal Semantic Search**: Perform natural language queries across all indexed data types simultaneously.
- **Contextual Retrieval**: Find relevant information based on the user's current situation, time, and location.
- **FAISS Vector Indexing**: High-performance similarity search for efficient data retrieval.

## 3. Behavioral Analytics
- **Pattern Recognition**: Identifying daily, weekly, and temporal routines.
- **Habit Analysis**: Tracking consistency in exercise, work, and personal activities.
- **Daily & Weekly Summaries**: Automatically generating comprehensive overviews of activity, achievements, and trends.
- **Day Rating System**: Quantifying daily productivity and balance on a 1-10 scale.

## 4. Proactive Nudging
- **Location-Based Nudges**: Providing contextual suggestions when the user is near important locations (e.g., gym, office, Michael Crouch Innovation Centre).
- **Conflict Detection**: Identifying schedule conflicts that might interfere with suggested activities.
- **Optimal Timing Suggestions**: Recommending the best times for activities (like gym sessions) based on calendar availability and historical patterns.

## 5. Intelligent Model Processing
- **Hugging Face Integration**: Leveraging state-of-the-art LLMs (e.g., Mistral-7B) for insight generation and synthesis.
- **WhiteCircle Quality Gates**: Automated validation to prevent hallucinations and ensure high-quality, factually-grounded responses.
- **Personal Insights**: Synthesizing data from multiple domains to provide holistic lifestyle recommendations.

## 6. Technical Capabilities
- **MCP Server Implementation**: Exposing tools, resources, and prompts via the Model Context Protocol.
- **HTTP API Bridge**: Integration-ready server for frontend applications.
- **Codebase Analysis**: Tools specifically designed to search, analyze, and suggest optimizations for software projects.
